Playing with the HTML5 range slider input

Basic input

<input type="range" />

Using range

Set min and max atribute to determine the range of possible values for the input.

<input type="range" min="0" max="100"/>

Result

Step atributte

Use value to set a default value and step sets or returns the value of the step attribute of the slider control.

<input type="range" min="0" value="10" max="100" step="10"/>
